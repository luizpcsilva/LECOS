Para usar o energygraph, primeiro faça o download de todos os arquivos e adicione-os à mesma pasta.
Em seguida, abra a pasta no seu terminal e execute o comando ```make```. 
Por fim, para rodar o programa, use ```./energygraph``` com permissões de root.

Caso queira parar a leitura, use <kbd>Ctrl</kbd> + <kbd>C</kbd> para visualizar os últimos segundos ou <kbd>Esc</kbd> para limpar o terminal.
